TypingSound

A Windows tray app that plays a pleasant sound on every keystroke — typewriter keystrokes, with a return bell on Enter.

⬇ Download (latest) View on GitHub

Windows 11 / portable, no install / no admin rights

Features

⌨️

Typewriter mode

A keystroke sound on every key, a carriage-return bell on Enter. Satisfying just to type.

📦

Portable

Self-contained with the runtime bundled. Unzip, double-click, done. Carry the whole folder anywhere.

🔇

Light & resident

Sits quietly in the system tray. Unnecessary heavy components are stripped at build time.

🔒

Privacy

Never handles which key you pressed. Nothing recorded, nothing sent, no network access.

Getting started

  1. Download the zip from the latest release.
  2. Unzip it anywhere you like.
  3. Double-click TypingSound.exe. It lives in the tray and starts playing right away.

Build from source

The .NET SDK and just are managed declaratively with mise.

mise install        # install the dotnet SDK and just per mise.toml
just build-all      # build the whole solution
just test           # Core unit tests
just dist           # create the distribution layout dist/TypingSound/
